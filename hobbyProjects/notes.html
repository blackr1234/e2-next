Overview
A note-editing Java program written in JavaFX framework that connects to a RethinkDB database (NoSQL, Document-oriented).
Project nature: Individual (hobby project)
Year: 2016
Functionalities:

 1. Note editing
 2. Account registration
 3. Online storage and document synchronization
 4. Note search

There is also another version that connects to a MySQL database.

* This tool can be installed on jPlatform.
