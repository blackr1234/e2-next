Software Version Checker

Overview
A JavaFX program coded with third-party API that simplifies the process of checking software versions on official websites of the software that I use.
Project nature: Individual (hobby project)
Year: 2017
When there is a newer version released, the software is highlighted.

Clicking on an item will open up the equivalent URL in the browser.

* This tool can be installed on jPlatform.
