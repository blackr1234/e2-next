Self-built PC Hardware Picker

Overview
Written in JavaFX. All catgory data and product information are retrieved from a self-built database with dynamic product categories.
Project nature: Individual (hobby project)
Year: 2016
The two latest versions can directly retrieve hardware product information from the Jumbo Computer website and the Centralfield website and provide handy functionalities:

 1. Save and open hardware lists
 2. Alert when product(s) are not found from the server
 3. Alert when the total price has fluctuated since last save

* These tools can be installed on jPlatform.
